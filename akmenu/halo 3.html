Theme Installation Instructions

  1. Download halo 3.7z.

  2. Open/Extract the .7z archive. You can use 7-Zip to do this.

  3. Inside the 7z you will find a folder called halo 3. Copy this folder to /__rpg/ui on your flashcartâ€™s SD card.

  4. Place the SD card back into your cart, and boot into the cart.

  5. Press the START key to open the AKMenu/WoodR4 start menu popup, then select Settings.

  6. Set the the theme to halo 3, then press A to save and restart to apply the theme.
